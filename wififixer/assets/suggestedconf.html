Configuring your Wifi Access Point

    Channels and Protocols

  • Protocol
    • If you can, set to g, b, a, or n. Mixed modes can be problematic.

  • Encryption
    • WPA2-AES is strongly recommended

  • Short Guard Interval
    • If you are dropping connection too often, try setting this to off.

  • Long Preamble
    • It's recommended you set this to enabled: it allows the phone's low power radio states to remain connected
