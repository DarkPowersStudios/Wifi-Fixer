Configuring your Wifi Access Point

Channels and Protocols:
  • Protocol
    • If you can, set to g, b, a, or n. Mixed modes can cause problems.

  • Encryption
    • WPA2-AES is strongly recommended
    WPA if WPA2-AES isn't available
    WEP 128 if WPA isn't available

  • Short Guard Interval (N only)
    •  If you are dropping connection too often, set this to off.

  • Long Preamble
    • Set this to enabled: it allows the phone's low power radio states to remain connected.

  • Channels
    • Channels 6 tends to be default.  If you are having interference problems (lots of drops, difficulty with initial connection) try setting to channel 1 or 11.

    Caveats:

  • Hidden SSID
    • Android will not operate correctly with hidden networks.  You must enable SSID broadcast for Android compatibility.

  • Ad Hoc
    • Non-infrastructure (peer to peer) wif networks aren't supported at this time by Android.
