Known Issues:

Task Killers

Important: Wifi Fixer's service can be rendered inoperable by task managers and auto task killers.

It is strongly recommended that you do not use Wifi Fixer with this kind of software, but if you must, add WifiFixerService and LogService to the allowed list.


Samsung Galaxy


Wifi may become unable to turn on without the message " Unable To Scan For Networks " displaying.

At this time wifi may become unusable. This is a known issue with the Samsung's build of Android.

It can be fixed either by rebooting the device or going into Settings:Privacy and doing a factory reset

Note: You will lose all your settings if you perform a factory reset.


Wifi Sleep Policy


On some devices, Wifi Sleep Policy appears to be ignored when the screen is off.

This is caused by wifi power save mode, which is not compatible with some wifi access points/routers.

There is a workaround labelled Nexus One Sleep Fix in Advanced Options which may help.

Note: The workaround may take 15-30 seconds for full network function.

Set Wifi Usability to Max Wifi before enabling the workaround, and set Wifi Sleep Policy to Never

Cyanogen Mod 6 and above have a driver fix for this issue:
Cyanogen Mod Homepage


Network Notifications

Because Wifi Fixer intercepts Wifi scans, it causes Network Notifications in wifi settings not to function.

Wifi Fixer has its own Network Notifications as an option in its preferences.


Known Networks

The new Known Networks slide-up panel may not show any networks when wifi is disabled.

The network list cannot be queried when wifi is off.


Other Wifi Apps

Wifi Fixer essentially takes full control of wifi, which may conflict with other apps that talk directly to the Android wifi manager. Conflicts have been minimized, but may still exist.

Hidden SSIDs

Android is not able to work properly with wifi networks using hidden SSIDs.

If your network is set to non-broadcast SSID, you will have to change it to broadcast.
