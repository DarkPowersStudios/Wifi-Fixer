Wifi Usability

  • Max Wifi
    • Disable Sleep and Wifi Lock are enabled. This uses the most battery, but preserves wifi connections even when the screen is off.

  • Balanced
    • Wifi Lock is enabled. Uses slightly more battery, but improves stability of wifi connection when the screen is on.

  • Max Battery
    • Disable Sleep and Wifi Lock are disabled. Uses the least battery while still maintaining wifi connection.


Note: Balanced is the default setting.
