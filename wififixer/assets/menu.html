Menu:

Toggle Logging


This menu option enables or disables logging.
Toggle Logging is not visible by default; to make it visible, use Preferences.

The icon will be red if logging is disabled, or green if logging is enabled.

Please note that logging does use more memory; leave it off until you need to log something for support.

The log can be found in your sdcard in /sdcard/data/org.wahtod.wififixer/wififixer_log.txt

Send Log

Send Log is not visible by default; to make it visible, use Preferences.

Send Log creates an email to the developer including some basic information about your phone, and attaches the log.

Preferences

When tapped, this will open the Preferences screen, which allows you to configure the app.
