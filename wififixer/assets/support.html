Support:

As an Open Source project, Wifi Fixer depends on you to help make it better.
Constructive feedback, ideas, and problem reports are always appreciated.
When emailing the developer, please try to be specific and descriptive.

Wifi Fixer has a logging feature which lets you send diagnostic logs directly to the developer with one tap.

The log's contents are helpful in diagnosing issues you might have.

By default both the logging feature and the menu items for the logging feature are disabled.

To enable the send log and enable logging menu items, check "Enable Log Menu" in Wifi Fixer's Logging preferences.

As of 0.9.0, Wifi Fixer will now record any exceptions (Force Closes) in the log. 
If you get a force close, please enable logging and send the log with an issue report.

NOTE:
Please leave logging on for at least 10 minutes.
 
When you send the log, Issue reports are important.  Please include time as well. 
(Some logs are rather large.  It would take too long to read through to find the issue without a report.)
