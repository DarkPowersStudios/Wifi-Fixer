Known Networks


At the bottom of the main activity screen is a slider widget which can be dragged up to reveal the Known Networks panel.

This panel will display a list of the configured networks from the device's Wifi settings.

When one of the known networks is in range, it will be highlighted yellow.

Tapping on a network in the Known Networks panel brings up a context menu with several options.

Some of these options will be greyed out if not currently selectable

  • Enable: flags network as available for connection
  • Disable: flags network as unavailable for connection
  • Connect Now: requests connection to network
  • Toggle Non-Managed: flags or unflags network for Wifi Fixer management

Note: Networks are managed by default.

If you find a network that is not compatible with Wifi Fixer's method of network checking, setting Non-Managed tells Wifi Fixer not to check or repair when connected to that network.
