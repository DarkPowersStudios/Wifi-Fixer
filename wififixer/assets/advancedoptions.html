Advanced Options




Wifi Usability
These options are automatically set by the Wifi Usability dialog in the main app, but can also be set here.

  • Enable Wifi Lock
    • Enables Wifi Lock, which tells Android to keep the Wifi radio awake. Uses slightly more battery.

  • Disable Sleep
    • When checked, Wifi Fixer will continue to monitor your connection and repair it when the screen is off. Uses moderately more battery.



Specialized Fixes


  • Wifi Sleep Policy
    • Tells the Wifi driver when to power down: see Known Issues

  • Disable Supplicant Fixes
    • Some Android 1.5 and 1.6 devices may operate better with this checked.

  • Nexus One Sleep Fix
    • For Nexus One and some other devices, if Wifi Sleep Policy is not working, may keep Wifi connected when the screen is off. See Known Issues.
