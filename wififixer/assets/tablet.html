Tablet Interface


On a Tablet, the screen is divided into four sections:

Portrait
  • Top Left
    • Contains the Status fragment.

  • Top Right
    • Contains the Log fragment.

  • Bottom Left
    • Contains the Known Networks fragment.

  • Bottom Right
    • Contains the     Local Networks fragment.


Landscape
  • Top Left
    • Contains the Status fragment.

  • Bottom Left
    • Contains the Log fragment.

  • Middle
    • Contains the Known Networks fragment.

  • Right
    • Contains the     Local Networks fragment.
