Preferences:

Disable Service

When checked, the background service will be disabled.

This allows you to run apps which may conflict with Wifi Fixer, or let you use an incompatible Wifi network.

The main screen will show Active when the service is running and Inactive when it's disabled, to remind you.


Set Wifi Usability

This is an easy way of setting some Advanced options.


  • Max Wifi
    • Turns on Wifi Lock and Disable Sleep, which maximizes your connection's stability but uses the most battery.

  • Balanced
    • Turns on Wifi Lock, which provides better wifi stability with slight battery drain.

  • Max Battery
    • Disables Wifi Lock and Disable Sleep, which minimizes battery use.

Notifications


Enable Notifications
When enabled, Wifi Fixer will use the Android Notification bar to alert you when it fixes a problem.


Network Notifications
When enabled, Wifi Fixer will notify you of open wifi networks


Advanced Options

Note that Set Wifi Usability changes some of these options automatically.


Enable Log Menu
By default the Logging menu options are hidden.

This option enables them, but does not turn logging on.

Note: Logging records some personal information, including time zone setting and names of configured networks.

The log may manually be emailed to the developer.


Enable Wifi Lock
When checked, causes wifi to hold a lock, which helps keep the connection from dropping.

A wifi lock will use slightly more battery.


Disable Sleep
Normally, Wifi Fixer goes to sleep when the screen is off, to conserve battery.

This option, when checked, keeps Wifi Fixer checking the connection even when the screen is off.

This will use moderately more battery.


Wifi Sleep Policy
Offers a convenient way to change Wifi Sleep Policy. Normally changed in Wifi Settings:Advanced


Disable Supplicant Fixes
If checked, this will disable supplicant fixes. An android 1.5 or 1.6 device may function better with this checked.


Nexus One Sleep Fix
If checked, enables Nexus One Sleep Policy Fix. Important: Wifi Lock and Disable Sleep must be on. Wifi Sleep Policy must be set to never.

Even then it may not work for you.


Widget Behavior

By default the widget will reassociate with the wifi access point, if available.

If checked, the widget will reset wifi by turning it off and then on. This may fix some problems the service cannot.


The widget can be added to the desktop by long-pressing then selecting it from the Widgets list.
