Preferences:

Disable Service

When checked, the background service will be disabled. This allows you to run apps which may conflict with Wifi Fixer, or disable it if you're using an incompatible Wifi network.
The main screen will show Active when the service is running and Inactive when it's disabled, to remind you.

Advanced Options

Note that Set Performance changes these automatically.
Enable Wifi Lock
When checked, causes wifi to hold a lock, which helps keep the connection from dropping. A wifi lock will use slightly more battery, though.
Disable Sleep
Normally, Wifi Fixer goes to sleep when the screen is off, to conserve battery. This option, when checked, keeps Wifi Fixer checking the connection even when the screen is off.
This will use slightly more battery.

Set Performance

This is an easy way of setting the Advanced options. Max Wifi turns on Wifi Lock and Disable Sleep, which maximizes your connection's stability.
Balanced turns on Wifi Lock. Max Battery disables Wifi Lock and Disable Sleep, which minimizes battery use.

Enable Notifications

When enabled, Wifi Fixer will use the Android Notification bar to alert you when it fixes a problem.

Enable Log Menu

By default the Logging menu options are hidden. This option enables them, but does not turn logging on. Note: Logging records some personal information, including time zone setting and names of configured networks. The log may manually be emailed to the developer.
